Hoe werkt GitHub Desktop?
1. Downloaden en Installeren
Surf via desktop.github.com naar de download-pagina en druk op de "Download for ..."-knop. Open het gecomprimeerd archief via uw Bestanden-app. Pak het uit in een folder dat u makkelijk kunt vinden en open het "GitHub Desktop"-bestand.
2. Git configureren
Log in met uw GitHub account, en configureer uw Git: Bewaar uw username en verander de e-mail naar uw persoonlijke, niet diegene die git zelf aanmaakte. Tik daarna op "Finish"
3. Uw Repository in GitHub Deskstop plaatsen
In het beginscherm, klik op "Clone a repository from the internet". U kunt zoeken achter uw Repository via GitHub.com, GitHub Enterprise, of via een URL. Kies uw repository en waar in uw computer het bewaard zal worden. Tik als laatste op "Clone", en u bent nu in de interface van uw Repository!
4. Functies van GitHub Desktop
Fetch Origin: Controleren of er een recente verandering werd gemaakt aan de originele code door iemand anders.
Pull Origin: Doet een update aan uw kopie van de Repository.
Changes: Detecteert alle veranderingen in de folder die u maakte.
Commit: Plaatst uw Changes in de kopie van de code.
Push Commit: Stuurt uw changes naar uw GitHub Repository.
History: Een tijdlijn van alles dat veranderd is aan de Repository (nieuw naar oud).
5. Slot: Waarom GitHub Desktop?
Wij vinden dat GitHub Desktop de aanbevolen manier is om aan uw Repository te werken. Het is gratis, officieel en tot slot simpel om te gebruiken en te navigeren. Snapt u het nog steeds niet? Dan zal de video met meer visuele uitleg u verder helpen.
