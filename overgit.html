Wat is GitHub?

GitHub is een platform voor softwareontwikkeling en versiebeheer. Het wordt wereldwijd gebruikt door developers om software, Apps, Website en dergelijken te beheren. GitHub levert features die je kunt gebruiken om gemakkelijk bugs op te lossen of updates te maken zonder dat je je broncode in gevaar brengt. Je kunt in GitHub ook kiezen om je code openbaar te maken zodat mensen je broncode kunnen downloaden en kunnen gebruiken. GitHub is ook een heel gemakkelijk platform als je met meerdere mensen aan 1 project wilt werken op een effeciente manier. Kortom GitHub is het ideale platform om als developer in een groot team te werken en professioneel je App, Website, ... te beheren op een gebruiksvriendelijke manier!
