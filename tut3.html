Wat is en hoe maak ik een Repository?
1. Repository Aanmaken
Om een Repository aan te maken op github, moet je eerst naar de + gaan op de rechterbovenhoek op Github, dan verschijnt er een dropdown-menu, klik op "New Repository", je komt dan terecht in een menu waarbij je repository kan maken.
2. Instellen van de Repository
Bij de instellingen, ga je eerst naar de "Repository Name', kies een memorable naam die je kan herinneren. Sla de descriptie over of schrijf er iets nuttig in. Zet je repository op Private omdat je hier het meeste van je schoolwerk doet. Voeg een README file toe, sla de license en .gitignore over. En gefelicticeerd, je hebt je eerste Repository ooit!
3. Waarom heb ik een repository nodig?
Een Repository is een plaats waar alle bestanden van je project bewaard worden en de versiegeschiedenis ervan, het functioneert als een soort “hoofdmap” voor je projecten. Zonder een Repository kan je trouwens niet veel doen. Het is van essentieel belang om er eentje te hebben.
